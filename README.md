## Pixel Hop (Web 2D Platformer)
This project now contains a single-window browser platformer game built with plain HTML, CSS, and JavaScript.

### Gameplay
- Move with `Arrow Left` / `Arrow Right` (or `A` / `D`)
- Jump with `Arrow Up`, `W`, or `Space`
- Press `R` to restart
- Collect all stars to win

### Run locally
```bash
python -m http.server 8000
```
Then open `http://localhost:8000`.
